Dokument

Trontalen gjev uttrykk for hovudlinene i arbeidsprogrammet til Regjeringa og politikken for det komande året.

Melding om rikets tilstand er ein gjennomgang av regjeringspolitikken det siste året, og omtalar kva som er gjort, og viktige framlegg.

Tiltredningserklæring er ei erklæring om hovudlinene i den politikken som den nye regjeringa vil føre.

Rapportar og planar er statusrapportar frå regjeringa og planlagde saker til Stortinget.

Veiledninger og brosjyrer er blant annet veiledninger fra Statsministerens kontor til departementene om statsråd og r-konferanser.

Proposisjonar og meldingar er dokument som blir lagd fram for Stortinget.

Høringar er forslag frå departementa som sendast til partar (offentlege og private institusjonar, organisasjonar og andre departement). Bakgrunnen er eit ønskje om å betre kunne vurdere dei økonomiske og administrative konsekvensane av offentlege tiltak.

NOU-ar Regjeringa eller eit departement set ned utval og arbeidsgrupper som greier ut ulike forhold i samfunnet. Ei utgreiing kan enten kome som NOU (Noregs offentlege utgreiingar) eller som ein rapport.
