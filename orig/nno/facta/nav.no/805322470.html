Funksjonsassistanse i arbeidslivet

Personar med sterk fysisk funksjonsnedsetjing kan søkje om funksjonsassistanse. Arbeidstakar/‑søkjaren nyttar kompetansen sin og får avlasting med praktiske oppgåver.

Eksempel på assistanse: leggje papir i skrivaren, lyfte eller hente tunge ting og hjelpe med å kle av og på. Funksjonsnedsetjinga må vere varig.

 

Kven kan søkje

  • Arbeidstakarar som er tilsette i ei verksemd. Det omfattar fast tilsette, vikarar, mellombels tilsette og personar som er tilsette med lønnstilskot (tidsbestemt og ‑ubestemt)
  • Sjølvstendig næringsdrivande
  • Frilansarar utan tilsetjingsforhold
  • Arbeidssøkjarar

 

Kva må gjerast

Ein person som har problem med å fungere i arbeidssituasjonen på grunn av sterk fysisk funksjonsnedsetjing, skal først og fremst snakke med den nærmaste leiaren for å undersøkje sjansane for tilrettelegging på arbeidsplassen. Det kan vere organisatorisk eller fysisk tilrettelegging, tekniske hjelpemiddel eller ombygging av utstyr eller lokale (sjå Hjelpemiddel og tilrettelegging av arbeidsplassen). Bedriftshelsetenesta er ein viktig støttespelar. Dersom det ikkje er nok, kan ein søkje om funksjonsassistanse. Søk på same skjemaet som tekniske hjelpemiddel eller på eige ark. Send søknaden til NAV Hjelpemiddelsentral i fylket.

Frilansarar eller personar utan arbeid kan kontakte hjelpemiddelsentralen eller NAV-kontoret for å få meir informasjon.

 

Kva blir dekt

Kor mange timar funksjonsassistanse den enkelte treng, blir vurdert individuelt. Assistenten kan vere ein kollega som blir frikjøpt i så og så mange timar, eller ein person som blir tilsett sjølvstendig. Arbeidsgivaren forskotterer lønn, arbeidsgivaravgift og sosiale utgifter. Det blir refundert fire gonger per år. Andre ordningar kan òg avtalast. Ein kan få funksjonsassistanse så lenge ein har behov for det, med eitt atterhald: Ordninga er avhengig av årleg løyve frå Stortinget og er ikkje rettsfest.
